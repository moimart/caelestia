# Keybinds Cheat Sheet

## Launcher & Shell

| Keys | What it does |
|------|-------------|
| `Super + Space` | Summon the app launcher. Your gateway to everything. |
| `Ctrl + Alt + Delete` | The holy trinity of "I need to log out RIGHT NOW." |
| `Ctrl + Alt + C` | Silence the notification swarm. Peace at last. |
| `Super + K` | Reveal all panels like pulling back a curtain. |
| `Super + L` | Lock screen. Trust nobody, not even yourself. |
| `Super + Alt + L` | Restore lock + restart shell. For when locking alone wasn't dramatic enough. |

## Workspaces

| Keys | What it does |
|------|-------------|
| `Ctrl + Left/Right` | Hop one workspace left or right. Quick lane change. |
| `Super + 1-9, 0` | Teleport to workspace 1-10. No stops in between. |
| `Ctrl + Super + 1-9, 0` | Jump to workspace *group* 1-10. Workspaces have workspaces. Inception. |
| `Ctrl + Super + Left/Right` | Slide to the next/previous workspace. The scenic route. |
| `Super + Page Up/Down` | Same thing, for those who love Page keys. You exist, apparently. |
| `Ctrl + Super + Scroll` | Skip 10 workspaces at a time. For the truly unhinged multitasker. |
| `Super + S` | Toggle the special workspace. Your secret drawer. |

## Moving Windows Between Workspaces

| Keys | What it does |
|------|-------------|
| `Super + Alt + 1-9, 0` | Yeet the active window to workspace 1-10. |
| `Ctrl + Super + Alt + 1-9, 0` | Yeet it to a workspace *group*. Maximum yeetage. |
| `Super + Alt + Page Up/Down` | Nudge the window one workspace over. Gentle yeet. |
| `Super + Alt + Scroll` | Mouse-powered window relocation. Lazy but effective. |
| `Ctrl + Super + Shift + Left/Right` | Shove the window left/right one workspace. With authority. |
| `Ctrl + Super + Shift + Up` | Banish window to the special workspace. Out of sight, out of mind. |
| `Ctrl + Super + Shift + Down` | Rescue window from special workspace back to the mortal realm. |
| `Super + Alt + S` | Another way to toss a window into special workspace. Options! |

## Window Focus & Movement

| Keys | What it does |
|------|-------------|
| `Super + Arrows` | Move focus in any direction. Your eyes, but for windows. |
| `Super + Shift + Left/Right` | Swap windows sideways. Musical chairs, window edition. |
| `Super + Shift + Up/Down` | Move window up/down. Vertical ambition. |
| `Super + Shift + Ctrl + Left/Right` | Move window left/right (no swap). Assertive repositioning. |
| `Super + LMB drag` | Grab and drag a window anywhere. Caveman mode. |
| `Super + RMB drag` | Resize a window by dragging. Stretch it like pizza dough. |
| `Super + Z (drag)` | Keyboard-initiated window drag. For mouse-key hybrids. |
| `Super + X (drag)` | Keyboard-initiated resize. Same energy, different shape. |

| Keys | What it does |
|------|-------------|
| `Super + W` | Close the active window. Goodbye, you served well. |
| `Super + F` | Fullscreen. Go big or go home. |
| `Super + Alt + F` | Fullscreen but keep the borders. Big, with boundaries. |

## Window Groups
